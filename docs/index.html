Chapter 1 Course Overview

This is the complete set of course materials for the Quantitative Genetics Graduate Education Module (GEM) at the University of Oregon for the Spring of 2020. It is written in Markdown so that it can be easily updated.

In book you will find nearly all the information you will need to complete the course.
